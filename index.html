Hello Git

    Commands
  • git clone path-to-repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый файл под контроль GIT
  • git commit -am "added new file index.html" - зафиксировать изменения и дать им комментарии
  • git push - отправить все закоммиченные изменение на сервер
  • git pull - запросить все изменения с сервера
